3.文学大纲

文学大纲

说明

主要参考《文学大纲》，再对比木心回忆录收录的名人，进行汇总。中世纪指公元5世纪-公元15世纪。

中世纪之前

  1. 荷马史诗：古希腊最早的一部史诗，产生于公元前8世纪
    • 《伊利亚特》：述说希腊联军统帅阿伽门农和大将阿基里斯之间的冲突、在特洛伊战争中十年发生的故事
    • 《奥德赛》述说伊塔卡岛国王奥德修斯攻克特洛伊后返回家乡，却在途中漂泊了十年的故事。
  2. 希腊神话：古希腊人的一系列神话故事。
    • 赫西俄德的《神谱》和《工作与时日》：早期希腊神话以及详细的族谱，民间传说
    • 抒情诗《史诗集成》
    • 悲剧故事（如阿伽门农和他的儿女，俄狄浦斯，伊阿宋和美狄亚等）
  3. 世界三大宗教：
    1. 圣经：犹太教只认《旧约》，基督教认《旧约》（约公元前2世纪）和《新约》（约公元2世纪）
      • 《旧约》：分摩西五经、历史书、诗歌智慧书和先知书四部分
      • 《新约》：分福音书、使徒行传、保罗书信、大公书信和启示录五部分
    2. 印度婆罗门教的《吠陀》：婆罗马（创造者）、委西奴（保存者）、及西瓦（破坏者）
      • 印度教和佛教都源于婆罗门教
      • 印度教具有民族性限制、因此普遍认为佛教比印度教更适合称为世界三大宗教之一
    3. 回教的《可兰经》（或称伊斯兰教的《古兰经》），只有一个神（阿拉），教徒被称为是穆斯林
  4. 印度史诗：
    1. 《摩诃婆罗多》：主要讲俱卢家族的两兄弟，持国和般度的后代，为争夺王位进行的斗争
    2. 《罗摩衍那》：主要讲述拘萨罗国王子罗摩和他妻子悉多的故事。
  5. 诗经和楚辞：约公元前3世纪
    1. 《诗经》：分为风（十五国风）、雅（音乐）、颂（宗庙祭祀用诗歌）
    2. 《楚辞》：是一种诗歌的总集，作者有屈原、宋玉
  6. 古希腊戏剧：这类分为悲剧、喜剧、薩堤爾剧，繁荣于公元前6世纪末至前4世纪初
  7. 西方哲学的奠基者：亚里士多德、柏拉图、苏格拉底
  8. 汉至晋的文学：
    1. 东汉诗人：班固、傅毅、张衡、蔡邕
    2. 司马迁的《史记》
    3. 《古诗十九首》
    4. 五言体叙事诗：《陌上桑》、《妇病行》、《孤儿行》、《古诗为焦仲卿妻作》
    5. 三曹
    6. 竹林七贤是：山涛、阮籍、刘伶、嵇康、向秀、阮咸、王戎
    7. 陶渊明（陶潜）

中世纪

  1. 欧洲：
    1. 圣·奥古斯丁：《忏悔录》被称为西方历史上第一部自传，概述了他年轻时犯下的罪过和他改信基督教。
    2. 德国的史诗：《尼伯龙根之歌》
    3. 西班牙史诗：《熙德之歌》
    4. 《列那狐传说》：辛辣地嘲讽了专制的国王、贪婪的贵族、愚蠢的教士等
    5. 但丁：《神曲》想象了与地狱、炼狱以及天堂中各种著名人物的对话
    6. 佛罗伦萨文学三杰：但丁、薄伽丘、彼特拉克
  2. 中国诗人：
    1. 诗人：李白、杜甫、张九龄、王维、孟浩然、刘禹锡、韩愈、柳宗元、李贺、白居易、杜牧、元微之
    2. 《文心雕龙》，中国第一部系统文艺理论巨著，也是一部理论批评著作
    3. 诗人：李煜、范仲淹、欧阳修、陆游、苏轼、辛弃疾、李清照、王安石、文天祥
  3. 中国戏曲：
    1. 董解元：《西厢诌弹词》，题材源于元稹的《会真记》
    2. 关汉卿：《窦娥冤》、《单刀会》
    3. 王实甫：《西厢记》源于董解元的
    4. 马致远：《汉宫秋》写王昭君远嫁的故事
    5. 刘东山：《娇红记》
    6. 白朴：《梧桐雨》写唐明皇和杨贵妃、《墙头马上》
    7. 《荆钗记》、《白兔记》、《拜月亭》、《杀狗记》、《琵琶记》
  4. 中国小说：
    1. 六朝时分为两类：神怪故事如《搜神记》、名人言行如《世说新语》
    2. 唐传奇恋爱故事小说：《霍小玉传》、《会真记》
    3. 唐传奇豪侠故事小说：《无双传》、《虬髯客传》
    4. 唐传奇神怪故事小说：《李娃传》、《南柯记》、《离魂记》、《枕中记》、《刘毅传》、《长恨歌传》
  5. 波斯的诗人
  6. 印度的戏曲
  7. 阿拉伯的诗人，其中有《天方夜谭》
  8. 日本文学：
    1. 诗歌：《万叶集》
    2. 散文：《古事记》、《源氏物语》、《枕草记》
    3. 韵文：《古今集》
    4. 《平家物语》：写于1371年，讲述在12世纪末源家和平家的斗争
    5. 鸭长明的《方丈记》，写于1212年
    6. 吉田兼好的《徒然草》，成于1331年

中国15世纪~19世纪

  1. 《今古奇观》：冯梦龙《三言》和凌濛初《二拍》的合辑
  2. 《东周列国志》
  3. 《隋唐志传》
  4. 《三国志》、《金瓶梅》、《西游记》、《水浒传》
  5. 汤显祖的《牡丹亭》、《还魂记》、《南柯记》
  6. 纳兰性德：词人
  7. 金圣叹：文艺评论家
  8. 孔尚任的《桃花扇》
  9. 《红楼梦》、《儒林外史》、《绿野仙踪》
  10. 李汝珍的《镜花缘》
  11. 陈森之的《品花宝鉴》
  12. 文康之的《儿女英雄传》
  13. 韩子云的《海上花列传》
  14. 沈复的《浮生六记》
  15. 张岱的《陶庵梦忆》
  16. 李渔的《闲情偶记》
  17. 明代三才子徐渭：字文长，画画创立青藤派，杂剧有《四声猿》
  18. 明代三才子杨慎：《升庵集》
  19. 明代三才子解缙：擅长狂草，《解学士集》
  20. 晚清四大谴责小说：
    • 吴沃尧的《二十年目睹怪现状》
    • 李佰元的《官场现形记》
    • 刘鹗的《老残游记》
    • 曾朴的《孽海花》
